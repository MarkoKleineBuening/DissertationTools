Error loading XML data

The browser was not able to load XML data from local files.

This is caused by security restrictions, which can be avoided / bypassed as follows:

  • Either start an HTTP server in the directory, where the XML files are located, e.g., using python -m SimpleHTTPServer, and use the URL http://localhost:8000/html/report.html (preferred solution), or
  • Enable loading of local files. In Google Chrome, e.g., this can be accomplished by starting the browser with the option --allow-file-access-from-files.
